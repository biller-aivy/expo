## Get started

System requirements:

- [Git](https://git-scm.com)
- [Node.js (LTS)](https://nodejs.org/en/)
- [Watchman](https://facebook.github.io/watchman/docs/install#buildinstall) for Linux or macOS users

---

We recommend creating a project with our default project template:

<Terminal cmd={['$ npx create-expo-app@latest']} />

> You can use another template by adding the `--template` flag.
